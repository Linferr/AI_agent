# Concepts

这份文档用于沉淀本项目的“术语表 / 概念边界 / 约定”，避免大家在同一个词上各自理解。

## Terms

- `LLM`：模型能力的“服务化调用”，关注输入输出、成本、延迟、可靠性。
- `RAG`：检索增强生成；核心目标是“回答受证据约束、可追溯”。
- `Tool Calling`：让 LLM 以结构化方式调用外部函数/服务，并把结果回填形成闭环。
- `Agent`：受控的多步循环（规划→执行→观察→终止），不是更强的模型。
- `Observability`：让一次请求的链路与关键指标可见（耗时、token、错误、命中等）。
- `Evaluation`：用固定用例/数据集稳定对比不同策略、Prompt、检索与工具方案。
